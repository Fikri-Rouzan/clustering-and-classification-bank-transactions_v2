# Clustering and Classification Bank Transactions (V2)

## 📌 Deskripsi

Proyek ini berfokus pada analisis data transaksi perbankan menggunakan metode klasterisasi dan klasifikasi berbasis machine learning. Sistem ini mengeksplorasi pola aktivitas keuangan untuk mengidentifikasi perilaku anomali serta mendukung deteksi potensi penipuan pada akun nasabah. Hasil akhir dari proyek ini ditujukan untuk memetakan profil perilaku transaksi secara terukur guna membantu meningkatkan keamanan finansial dan mempermudah evaluasi risiko berkala.

---

## 💾 Dataset

Dataset yang digunakan dalam proyek ini memuat 2.512 sampel data transaksi perbankan historis yang menyajikan rincian pola aktivitas keuangan dan demografi nasabah. Atribut yang tersedia meliputi pengidentifikasi unik transaksi dan akun, nilai nominal, tipe transaksi, lokasi geografis, alamat protokol internet, kanal akses, serta durasi aktivitas. Selain itu, terdapat parameter pelengkap seperti usia, profesi pemilik akun, saldo akhir, frekuensi transaksi harian, dan jumlah upaya masuk akun yang digunakan sebagai parameter input untuk kebutuhan analisis anomali serta deteksi potensi penipuan.
